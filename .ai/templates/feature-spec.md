# 功能规格书模板

> **流程位置**：阶段② 接口合并（分支A），由 AI 根据 Swagger + PRD 合并生成
> **输出路径**：`specs/[feature]/feature-spec.md`

## 标注规范

| 来源标记 | 含义     | 差异状态  | 含义                       |
| -------- | -------- | --------- | -------------------------- |
| `[S]`    | Swagger  | ✅ 一致   | 双方定义吻合               |
| `[P]`    | PRD      | ⚠️ 需确认 | 仅单方存在或有差异但不阻塞 |
| `[M]`    | 合并推断 | 🔴 冲突   | 双方定义矛盾，阻塞开发     |
| `[S][P]` | 双方一致 |           |                            |
